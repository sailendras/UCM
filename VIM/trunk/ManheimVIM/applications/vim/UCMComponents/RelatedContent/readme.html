RelatedContent Component for 10g Content Server

Date: March 23, 2009
Component Version: 10g 2009_03_23 (build 16)
Product and Version: Oracle Content Server 10.1.3.3.1 + updates
Oracle Content Server 10.1.3.3.2 + updates

Component Information:

This component allows you to link content items together to form different relations.

For more information see the file related_content_80.pdf contained in the root directory of the component.

This is an upgrade of the RmaLinks component.

Installation Instructions:

Warning: This update involves installing a new version of a component. Oracle recommends you perform all necessary backups prior to applying this update. For additional information see the Content Server Getting Started Guide which contains information on System Maintenance and Backup Strategies.

  1. Download the component file.
  2. The file is an Oracle UCM component. Use the Component Wizard or the Admin Server Component Manager to install and enable the component.
  3. Restart the Content Server.

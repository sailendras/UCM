IdocScript Test Page

This page demonstrates that custom IdocScript can be executed in the same way as normal IdocScript. The Java Class "NewScriptExtensions.java" and the file "new_idoc_script_extensions.htm" demonstrate how to modify or add functionality to IdocScript with a component. Note that the 'log' function dumps output to the console, so you will need to 'View Server Output' from the Admin Server to see it.

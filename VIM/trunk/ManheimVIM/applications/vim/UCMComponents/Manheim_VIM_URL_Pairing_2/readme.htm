HOWTO COMPONENTS

DataAccess

This component enables the user to alter a table in the database from a template page. The first time this component is run, it will create and prefill the database table if needed.

This component could be modified into any kind of component that utilizes simple data entry. It can also be used as a template for any component that requires a modification to the database.

This component demonstrates:

  • Custom Templates
  • Custom Services
  • Custom Queries
  • Filters with custom Java code
  • Executing SQL from Java

Installation

This component will be installed when you install the component bundle "HowToComponents.zip". If you have not done so already, use the Component Wizard or the Component Installer on the Admin Server, and install the bundle "HowToComponents.zip." Then enable this component, and install the content server.

After installing the bundle, there will be the file "DataAccess.zip" that can be used to install this component seperately.

After restarting the content server, you should either see a drop down menu, or a Tray, called 'HowToComponents', in which you will see a link called 'DataAccess'. Select this link to take you to a page that enables you to modify the system database.

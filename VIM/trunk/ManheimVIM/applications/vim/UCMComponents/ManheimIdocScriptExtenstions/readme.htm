HOWTO COMPONENTS

NewIdocScript

This component demonstrates how to create custom IdocScript with a component. This is useful when you wish to be able to execute custom Java code frequently, on any template page.

Some examples of new variables include: an uppercase version of the user's full name, the current day of the week, a check for if the date is even, and a check for if the current user has a particular security clearance.

Some examples of new functions include: mathematical functions that cannot be performed simply with IdocScript, string sorting functions, or cryptographically strong hashes of strings.

This component demonstrates:

  • Custom Templates
  • New IdocScript functions with custom Java code

Installation

This component will be installed when you install the component bundle "HowToComponents.zip". If you have not done so already, use the Component Wizard or the Component Installer on the Admin Server, and install the bundle "HowToComponents.zip." Then enable this component, and install the content server.

After installing the bundle, there will be the file "NewIdocScript.zip" that can be used to install this component seperately.

After restarting the content server, you should either see a drop down menu, or a Tray, called 'HowToComponents', in which you will see a link called 'NewIdocScript'. Select this link to take you to a page that demonstrates the new IdocScript functions.

ScheduledTimeEvents Component

This component allows you to create custom events that will be managed by the Content Server's scheduled events. The component also allows you to specify how often and at what time of day you would like the event to run. See the ScheduledTimeEventsTest component for an example of how to use this component.

Requirements

Content Server Compatible Version: 7.5.x and later
